# Sheikh Ishtiaq - The Growth Strategist

A full-stack personal brand website built with the SERN stack:

- **Supabase** - Database and authentication
- **Express.js** - API backend
- **React** - Frontend framework
- **Node.js** - Runtime environment

## Project Structure

- `client/` - Public-facing React website (Vite)
- `server/` - Express.js API backend
- `admin/` - Admin dashboard React app

## Getting Started

1. Install dependencies in all three folders
2. Set up environment variables in each .env file
3. Start development servers:
   - Client: `npm run dev` (port 5173)
   - Server: `npm run dev` (port 5000)
   - Admin: `npm run dev` (port 5174)
